# Login Layout

A Next.js authentication flow with mobile verification, user details, and dashboard.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Live at [layout-login.vercel.app](https://layout-login.vercel.app)
